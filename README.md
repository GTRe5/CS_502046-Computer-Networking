# 📡 Cisco Packet Tracer Compatibility Notice

## ⚠️ Required Version

This repository contains network simulation files that are **only compatible with Cisco Packet Tracer version *9.0.0* or later**.

Using an older version may result in:

* Inability to open `.pka` files
* Missing configurations or features
* Errors during simulation

---

## ✅ Recommendation

Please ensure you are using the correct version of **Cisco Packet Tracer** before opening any files in this repository.

## 🏷️ File Naming Convention

All lab files in this repository follow a consistent naming format:

```
ID_NAME - LAB's Name.pka
```

👉 Example:

```
502046_PhamQuocHung - VLAN Configuration.pka
```

### 📌 Rules:

* `ID`: Your student ID
* `NAME`: Your profile name (same as `Profile Name` in Cisco Packet Tracer)
* `LAB's Name`: Descriptive name of the lab
